---
description: Структура и управление пакетами вопросов (JSON) для игры
---
Игра поддерживает внешние пакеты вопросов, которые должны размещаться в директории `src/data/packages/`. Главный файл будет подключен как state.

## Формат пакета (JSON)
Данные вопросов должны сохраняться в строгом формате:

```json
{
  "id": "package_1",
  "name": "Стандартный Пакет Игр",
  "rounds": [
    {
      "roundName": "Раунд 1",
      "categories": [
        {
          "categoryName": "Космос",
          "questions": [
            { "price": 100, "text": "Самая большая планета?", "answer": "Юпитер" },
            { "price": 200, "text": "Красная планета?", "answer": "Марс" },
            { "price": 300, "text": "Спутник Земли?", "answer": "Луна" }
          ]
        }
      ]
    }
  ]
}
```
**Действия AI при добавлении вопросов:**
При разработке пакетов всегда проверяйте структуру TypeScript типов (в папке `types/` или `store/`) и валидируйте создаваемый JSON.
